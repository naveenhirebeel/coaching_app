# CoachingBuddy — Feature Suggestions

## 1. Fee Management
Set monthly fees per batch, record payments per student, flag overdue accounts. Add Telegram reminders to parents for unpaid fees.

## 2. Student Performance / Marks Tracking
Record test/exam scores per student per batch. Show summaries to admin and optionally share with parents via Telegram.

## 3. Parent-Facing Portal or Telegram Menu
Let parents check their child's attendance history and upcoming schedule via Telegram bot commands or a token-based web page.

## 4. Leave / Holiday Calendar
Admin-managed calendar of planned holidays. Auto-skip attendance on those days and send advance reminders to parents.

## 5. Multi-Batch Students
Students can belong to multiple batches (many-to-many). Currently limited to one batch per student in the schema.

## 6. Attendance Correction / Edit
Allow teachers to correct same-day attendance entries and admins to fix historical records. Corrections tracked in the existing audit log.

## 7. WhatsApp Notification Support
Add WhatsApp Business API (Twilio, Interakt) as an alternative to Telegram, targeting institutes where Telegram adoption is low.

## 8. Automated Weekly Admin Digest
Weekly Telegram or email summary to the admin: institute-wide attendance %, students below threshold, classes held vs scheduled.
